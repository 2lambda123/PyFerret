Ferret v5.70

Welcome to the MAC OS X download page for Ferret.

Ferret v570 beta was built through cooperative support by Andy Jacobson, then a researcher at Princeton University.

The ferret.tgz tar file is located here, ferret.tgz

Note that to use the install script, you will need to rename the tar file:

   mv fer_executables_v570alpha.tar.Z fer_executables.tar.Z

Here is a useful mail message from the email list about the tools Fpalette and Fpattern http://www.pmel.noaa.gov/maillists/tmap/ferret_users/fu_2007/msg00522.html

As the MAC OS X port evolves and becomes more stable, we will begin to host the MAC OS X executables here at PMEL. Until then, if you have any questions, or if you have trouble getting the executables, please email us at oar.pmel.contact_ferret@noaa.gov .

Big thanks go to Andy for making Mac Ferret a reality for so long!
