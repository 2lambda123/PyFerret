PyFerret Known Issues Addenda

The following describes recently discovered issues with PyFerret that are not listed in PyFerret Known Issues and Release Notes
The recently discovered issues listed here are those specific to PyFerret and does not include issues with the Ferret engine contained within PyFerret.


The Ferret "where" function does not work properly in PyFerret.
The graphics input function, which returns coordinates of a mouse-click, has yet to be implemented for the new graphics engines in PyFerret. As a result, the Ferret "where" function returns random coordinates without any interaction with the displayed window.
CTRL-C exits PyFerret
In Ferret, when CTRL-C is pressed, any pending activity in Ferret is stopped and the user is returned to the Ferret prompt. In PyFerret, CTRL-C causes the program to exit completely. This is due to Python capturing and reacting to the CTRL-C; thus Ferret functionality does not see the CTRL-C.
