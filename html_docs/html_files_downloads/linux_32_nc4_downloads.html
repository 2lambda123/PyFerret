Downloading Ferret for 32-bit Linux

Ferret v6.8

Ferret built on Red Hat Enterprise Linux Client release 5.6 (Tikanga), with gfortran (GCC) 4.2 It does not run on Linux Red Hat 3 or 4 (RedHat does not provide compatibility libararies to run software built on newer operating systems.)

NOTES for v6.8 and higher

  • If you are upgrading from a version lower than v6.8, the environment and external functions must be re-installed for this release.
  • If you have your own external functions, you must re-compile them using gfortran and the flags for compiling code to run as double precision. See the V6.8 release notes and the External Functions documentation pages.
