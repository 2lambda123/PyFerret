Downloading Ferret for Linux: statically linked

Ferret v6.5

Note: We have had reports the Ferret RedHat built with gcc/g77 has successfully been installed on SuSe 8.x systems, and also on Fedora Core 3 (FC3). You might try downloading the RedHat 32-bit executables first, and come back to the statically-linked executable if you run into incompatibilities with libraries.
