Installing Ferret

License Agreement

FERRET is available in binary for each of the systems listed below.

For the core supported Ferret binaries, here is information about shared-object libraries and Ferret executables.

Core Supported Operating Systems:

The release includes executables for linux RedHat 5, 32-bit and 64-bit; and one built under 64-bit and 32-bit linux RedHat 6, which should run on many of the other, free linux operating systems. Using compatibility libraries and installing the RedHat 5 executables will work as well. These are linked with NetCDF-4 libraries, with OPeNDAP dataset access enabled.

  • 32-bit Linux (RedHat 5) v6.81
  • 64-bit Linux (RedHat 5) v6.81
  • 64-bit Linux (RedHat 6 and equivalent) v6.81. Best for the various free Unix operating systems
  • 32-bit Linux (RedHat 6 and equivalent) v6.81. Best for the various free Unix operating systems
  • Mac OS X Intel v6.81 For Leopard(10.5), Snow Leopard(10.6) and Lion(10.7)

    • Ferret Source Code Downloads v6.81

    		 PLEASE NOTE! An alternative for running Ferret under Windows: We have had good successes running Ferret under the (free) Virtual Box virtualization environment. Install Virtual Box and then intstall one of the available free and well supported Unix platforms such as Ubuntu. This gives you an environment where you can install and run a Ferret executable for Linux. It also means you can run the latest Ferret releases rather than the old versions available for Windows. See the WINDOWS README file for more details.
