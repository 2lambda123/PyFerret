Cooperative Support of Ferret

 

Cooperatively supported operating systems are operating systems to which the Ferret developers do not have direct access. Ferret is supported on these systems through effort contributed by the Ferret user community -- individual sites who wish to have Ferret on these systems have volunteered to build it. The executables that they have created are available as cooperatively supported operating systems.

If you would like to have Ferret available on a system on which it is not currently, please write us at oar.pmel.contact_ferret@noaa.gov
