Downloading Ferret source code

Ferret v6.72

The Ferret Source code may be downloaded here.

The source code contains a README file with instructions for building it in the top-level directory.

As noted in the README file, if you build Ferret for a system other than linux 32-bit and 64-bit, we request that you send us the executables so that they may be shared with other users as one of our cooperatively supported operating systems.
