Downloading Ferret for IBM AIX

Ferret v6.301

Because IBM AIX is one of the Cooperatively Supported operating systems platforms, we have to rely on users to port Ferret to this platform. Thanks go to Hendrik Merx of the Max Planck Institut in completing the port of Ferret to this platform.

Ferret built on AIX.5.3.

Installation Instructions

  • Installation and update guide

Note: The executables tar file includes two Ferret executables: 

ferret_c.NetCDF
ferret_c.OPeNDAP
The ferret_c.NetCDF binary is linked against the NetCDF 4.0.1 library; ferret_c.OpeNDAP against libdap-3.8.2 and libnc-dap-3.7.3. The excutable with NetCDF 4.0.1 is included if you are interested in testing it; these libraries are quite new. They include DAP access that enables remote-file access, and this will in the future replace the OPeNDAP libraries for netCDF and remote-DAP dataset access in Ferret, but it is still in under final development. The name "ferret" is soft-linked to the ferret executable linked with the OPeNDAP libraries and so by default it will be installed as the Ferret executable.

The external functions cabability desn't work with these executables. Please unset the environment variable FER_EXTERNAL_FUNCTIONS, or set it to point to an empty directory.
