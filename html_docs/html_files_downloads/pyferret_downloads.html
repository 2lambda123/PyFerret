PyFerret is available pre-built as gzipped tar files for each of the systems listed below. The source code for PyFerret is also available as a gzipped tar file. See Installing or Building PyFerret for information on installing or building PyFerret from these files. At this time PyFerret cannot be built to run directly on Windows. See this WINDOWS_README file for more information on using a Linux version of PyFerret in a Linux virtual machine running under Windows.

This PyFerret 0.0.9 (beta) release, which encapsulates version 6.8.1 of Ferret, includes binaries for:

  • RedHat Enterprise Linux 5 (RHEL5), 32-bit and 64-bit, which using Python 2.6
  • RedHat Enterprise Linux 6 (RHEL6) and Ubuntu-10.10, 32-bit and 64-bit, which uses Python 2.6
  • Ubuntu-12.04, 32-bit and 64-bit, which uses Python 2.7
For many of the other, free Linux operating systems, the RHEL6/Ubuntu-10.10 or Ubuntu-12.04 versions (depending on whether it uses Python 2.6 or Python 2.7) should work. These binaries are (statically) linked with NetCDF-4.2 and HDF5-1.8.8 libraries, with OPeNDAP dataset access enabled.

PLEASE NOTE: There is no username or password required for downloading Ferret. If you see a pop-up window asking for such information, it is likely that your browser is not configured for anonymous FTP downloading. Either configure your browser for anonymous FTP, or try entering anonymous for the user ID and your email address for the password.

For installing PyFerret using the pre-built files, download both the PyFerret environment files tarball and the appropriate PyFerret binaries tarball. For building PyFerret from source code, you only need to download the PyFerret source tarball.

  • PyFerret 0.0.9 enviroment files (all platforms)
  • PyFerret 0.0.9 binaries for RHEL5 32-bit using Python 2.6
  • PyFerret 0.0.9 binaries for RHEL5 64-bit using Python 2.6
  • PyFerret 0.0.9 binaries for RHEL6/Ubuntu-10.10 32-bit using Python 2.6
  • PyFerret 0.0.9 binaries for RHEL6/Ubuntu-10.10 64-bit using Python 2.6
  • PyFerret 0.0.9 binaries for Ubuntu-12.04 32-bit using Python 2.7
  • PyFerret 0.0.9 binaries for Ubuntu-12.04 64-bit using Python 2.7
  • PyFerret 0.0.9 Source (all platforms)
