Ferret Installation Files


These are the three files a new user needs:

fer_executables.tar.gz 

	this compressed file contains the Ferret executable files

fer_environment.tar.gz 
	
	this compressed file contains all of the support files Ferret
	will need
	
fer_dsets.tar.gz 

	this compressed file contains demonstration data for
	review using Ferret
